# Comfortable YouTube

Make YouTube be more comfortable to use

---

Repository: <https://github.com/01101sam/Comfortable-Youtube>

Features:
- Skip ads (Only in watch page)
- Remove "Are you there" notify box
- Audio mode
- Block YouTube tracing
