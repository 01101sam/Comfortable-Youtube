# Comfortable YouTube

Make YouTube be more comfortable to use

---

Repository: <https://github.com/01101sam/Comfortable-Youtube>

Features:
- Remove all ads
- Remove "Are you there" prompt
- Audio mode
- Remove YouTube tracing
- Restore Download feature for normal user
